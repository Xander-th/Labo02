Stap 2

  1. Voel normalize.css toe via een link-tag in de head

  2. Voeg een Google font toe in de head (Montserrat 300, 500 & 800; normaal en italic)
  3. Koppel je eigen styles.css an je HTML

Stap 3: css

  1. 'corrigeer' de box-sizing property van alle elementen en van alle ::before en ::after pseudo-elementen.
    Zie theorie!
  2. Maak een body selector en pas de background color aan naar #fffdff,
    pas de font-family aan naar Montserrat en vergeet je fallback font niet!
    Zet de line height op 1.6 en de margin op 0

Stap 4: Vragen

Noteer het antwoord op deze drie vragen (het is wellicht interessant om ook telkens de vraag te herhalen in een heading van niveau 4...)

  1. Wat doet normalize.css

    2. Normalize.css zorgt er voor dat alle html en css code doorheen verschillende browsers zo goed als gelijk kan behandeld worden en dus gelijkaardige resultaten vertoont doorheen alle soorten browsers

  2. Wat is een fallback font, wat doet het, welk heb je gekozen en waarom?

    3. Een fallback font zorgt er voor, moest het gekozen font niet beschikbaar zijn dat er nog een back-up font aanwezig is die standaard in html verwerkt zit, ik heb hiervoor voor Arial of 'sans-serif' gekozen. Dit is een font dat veel gebruikt is en vooral makkelijk toegankelijk is

  3. Waar had je bij deze oefening moeite mee?

    4. De code van de Montserrat font deftig kunnen linken in de html, het was even zoeken of alles van lijndikte tot italic in 1 zat of ik meerdere Montserrat fonts moest importeren.
